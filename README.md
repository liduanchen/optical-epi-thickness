# EPI-Vision | 光学外延层厚度智能分析平台

> **企业级 Qt 界面 · TMM 精确拟合 · 实时物理动画 · 3D 结构可视化 · Manim 视频渲染**

一个专为半导体外延层（Epitaxy Layer）光学测厚设计的**一站式专业分析平台**，支持峰值间距法与传输矩阵法（TMM），提供完整物理可视化与批量数据挖掘功能。

---

## ✨ 主要特性

* **企业级深色拟物界面**（全面适配 PyQt6 / PySide6）
* **双核心算法引擎**：
    * **峰值间距法**：用于快速估算厚度。
    * **TMM 传输矩阵法**：s/p 偏振精确计算，支持联合反演。
* **智能反演模式**：
    * **固定折射率**：仅反演厚度。
    * **联合反演**：同时反演厚度与外延层折射率 (`n_film`)。
* **丰富实时可视化**：
    * **干涉光谱**：实验曲线 + 理论拟合 + 峰标记。
    * **光路动画** (`EpitaxyRayWidget`)：真实法布里-珀罗多次反射与流动激光效果。
    * **3D 多层膜结构** (`Structure3DWidget`)：厚度与入射角实时联动。
* **KPI 实时看板**：直观展示厚度、峰值数、Δν、拟合置信度及当前算法类型。
* **一键生成教学视频**：无缝对接 Manim 渲染引擎。
* **高效数据管理**：内置 SQLite 数据库，支持 Excel / CSV 批量导入。
* **批量规律学习**：自动生成“角度-厚度”及“角度-n_film”规律图表与 CSV 结果。

## 📖 详细使用指南

### 1. 数据导入（数据库接入）
* **支持文件格式**：`.xlsx`、`.xls`、`.csv`
* **必填字段**：
    * **材料名**（如：SiC、GaN、Si、GaAs）
    * **入射角度**（单位：度）
* **数据格式要求**（文件内）：
    * 第 1 列：波数（Wavenumber，单位 cm⁻¹）
    * 第 2 列：反射率（Reflectance，单位 %）
* *提示：可勾选“导入前清空同材料同角度历史数据”以避免数据重复。*

**内置材料默认折射率参考**：

| 材料 | `n_film` (外延层) | `n_sub` (衬底) |
| :--- | :--- | :--- |
| **SiC** | 2.60 | 2.55 |
| **Si** | 3.40 | 3.55 |
| **GaN** | 2.35 | 2.30 |
| **GaAs** | 3.30 | 3.45 |

### 2. 参数设置说明

| 参数 | 说明 | 推荐设置 |
| :--- | :--- | :--- |
| **数据集** | 已导入的 `材料 - 角度` | **必选** |
| **分析起点 (cm⁻¹)** | 光谱计算的起始波数 | 自动智能推荐 |
| **算法** | 选择 峰值间距法 或 TMM拟合法 | TMM（更精确） |
| **反演模式** | 选择 固定折射率 或 联合反演 | 若需探究折射率请选“联合反演” |
| **`n_film` / `n_sub`** | 外延层 / 衬底 折射率 | 自动预填（支持手动修改） |
| **n范围 ~ 峰距** | 联合反演时的 `n_film` 搜索范围与峰值最小间距 | 默认 1.80~4.20，峰距 30 |

### 3. 执行分析
* **一键自动分析**：系统智能选择最佳分析起点并立即运行。
* **执行分析**：严格使用当前面板上的参数手动运行。
* **结果呈现**：
    * **干涉光谱 (Tab)**：实验曲线（青色）+ 理论拟合（橙色）+ 峰标记（红色）。
    * **光路动画 (Tab)**：实时展现多次反射物理过程与流动激光。
    * **3D 多层膜 (Tab)**：直观展现厚度与角度联动的 3D 结构模型。

### 4. 生成 Manim 教学视频
分析成功后，点击 **“生成 Manim 视频”**：
1. 程序将自动生成 `temp_render_data.json`。
2. 自动调用 Manim 进行渲染（默认采用 `-pql` 快速模式）。
3. 视频将输出至：`media/videos/ThicknessAnimation/` 目录。

### 5. 批量规律学习
点击 **“批量规律学习”**，软件会自动遍历数据库中所有的光谱数据集，执行 TMM 联合反演，并输出：
* `batch_optical_patterns.csv`（完整分析表格）
* `batch_optical_patterns.png`（角度-厚度与角度-n_film 规律散点图）

## 🔬 技术实现亮点

* **核心算法**：结合 `scipy.signal.find_peaks` 寻峰逻辑与完整的 TMM 传输矩阵理论（严格区分 s/p 偏振态）。
* **可视化引擎**：
    * 自研 `EpitaxyRayWidget` 组件：实现动态多次反射的纯代码级动画。
    * 自研 `Structure3DWidget` 组件：基于 Matplotlib 3D 引擎。
* **底层架构**：SQLite3 自动建表与持久化数据管理。
* **UI 体验**：采用企业级 QSS 样式表，实现磨砂玻璃拟物风格与光影投影效果。

## ❓ 常见问题与解决 (FAQ)

* **Q1：启动时提示缺少 PyQt6 模块？**
    * A：请在终端执行 `pip install PyQt6` 或 `pip install PySide6`。
* **Q2：导入的数据库保存在哪里？**
    * A：默认保存在 `%LOCALAPPDATA%\EPI-Vision\optical_data.db`（若需要完全重置软件，可手动删除该文件）。
* **Q3：点击生成 Manim 视频后渲染失败？**
    * A：请检查 `manim_engine.py` 是否在主程序同级目录，并确保系统已正确安装 `manim` 库及其底层依赖（如 FFmpeg、LaTeX）。
* **Q4：为什么有时候检测不到峰值，或者拟合置信度很低？**
    * A：可以尝试在左侧面板适当**降低“分析起点”**，或者**调小“峰值间距”**参数，以适应不同材质的干涉周期。
* **Q5：在高分辨率（高 DPI）屏幕下，界面字体发虚或过小？**
    * A：右键点击 Python 可执行文件 -> 属性 -> 兼容性 -> 更改高 DPI 设置 -> 勾选“替代高 DPI 缩放行为”，并由“系统”执行。
